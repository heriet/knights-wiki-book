# キャラクター

騎士団・所属別。  
各騎士団内ではランク順（将軍→士官→その他メンバー）、同ランク内では年齢順で並べてます。  

一部、情報不足などの場合は、いちばん下に並べています。気づいたものから加筆・修正していってください。
