# 追放

キャラクターを追放することで、ユニットクラスとLvに対応した武器を入手できる。  
例えば、無属性のデュエリストのLVが

- 7～17の場合、エストック
- 18～26の場合、ハニーソード
- 27～34の場合、ドランゴンキラー
- 35～の場合、グランセイバー

となる
